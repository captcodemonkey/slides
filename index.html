Why Lighthouse?

  • Google: (They know a few things about search and web)
  • Free and simple: in Chrome DevTools since Chrome 60
  • It works: local, firewall, if you can see it lighthouse can see it
  • Extensible: standalone CLI
    [ npm i -g lighthouse]
  • Brand performance and competition

How does it work?

When running tests, Lighthouse by default runs from a cold cache (no files in cache), emulating 3G network speeds and average mobile device hardware by throttling the network and slowing down the CPU by 4x the machine's default speed. (To attempt to emulate real-world browsing conditions on average mobile hardware)
